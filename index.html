Trouver ses fichiers de police

Windows Ouvrez C:\Windows\Fonts\ dans l'Explorateur.
Clic droit sur une police → Propriétés pour voir le nom de fichier (.ttf / .otf).
macOS Via l'app Livre des polices (Finder → Applications).
Fichiers dans /Library/Fonts/ ou ~/Library/Fonts/.
Linux Dossiers /usr/share/fonts/ ou ~/.local/share/fonts/.
Commande : fc-list pour lister toutes les polices installées.
